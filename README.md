# HR Attrition Analysis Dashboard (Power BI)

## Project Overview
This Power BI dashboard analyzes employee attrition patterns using HR data. The goal is to identify key factors contributing to employee turnover and provide actionable insights for HR management.

## Key Metrics
• Overall Attrition Rate: 16.1%  
• Key drivers include overtime, job satisfaction, and income level.

## Dashboard Features
• Executive summary view of attrition metrics  
• Attrition breakdown by job role and department  
• Impact of overtime on employee turnover  
• Analysis of job satisfaction and work-life balance  

## Key Insights
• Employees working overtime show significantly higher attrition rates.  
• Sales Representatives experience the highest turnover risk.  
• Lower income brackets show disproportionately higher exit rates.  
• Low job satisfaction strongly correlates with employee attrition.
